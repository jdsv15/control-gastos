Una herramienta creada para gestionar tus finanzas personales directamente desde el navegador. Sin registros complicados ni configuraciones pesadas.

*¿Qué puedes hacer?*
Registrar movimientos detallados: Anota tus ingresos y gastos de forma rápida, asignando una fecha y una categoría específica a cada uno.

- Gestión por pestañas: Navega de forma limpia entre el formulario de "Nueva Transacción", un registro temporal de tu "Sesión Actual" y tu "Historial Global".

- Ver tu balance y alertas: Consulta cuánto dinero tienes disponible. El sistema cambiará a rojo y te mostrará una alerta de precaución si tus gastos exceden tus ingresos.

- Filtrar por categorías: Revisa en qué estás gastando más aislando los datos por categoría. El balance y el gráfico se actualizan mágicamente en tiempo real según lo que filtres.

- Análisis visual y estadístico: Incluye un gráfico de barras interactivo para entender tu flujo de dinero, además de un panel automático que detecta tu "Max Ingreso" y "Max Gasto".

- Tus datos están a salvo: Todo se guarda en el almacenamiento local de tu navegador (LocalStorage), así que no pierdes tu historial principal si cierras la pestaña.

*Cómo probarlo* (Sin instalar nada):

Esta aplicación está construida con tecnología web pura (HTML, CSS y JavaScript), lo que significa que es totalmente portable:

- Descarga o clona la carpeta del proyecto.

- Busca el archivo index.html.

- Haz doble clic en el archivo y se abrirá en el navegador.

*Un vistazo rápido*
css/: Estilos y diseño visual responsivo.

js/: Lógica de la aplicación dividida en controladores independientes (Store.js para almacenamiento, UI.js para la interfaz y App.js para la inicialización y eventos).

index.html: Estructura principal de la aplicación, formularios y contenedores de gráficos.
